# Simple Interest Calculator

A simple command-line application that calculates simple interest based on user-provided principal, rate of interest, and time period. This project demonstrates basic shell scripting and user input handling.

## Installation

To install the Simple Interest Calculator, follow these steps:

1. Clone the repository to your local machine:
git clone https://github.com/Hayman04/simple-interest-calculator.git

2. Navigate to the project directory:
cd simple-interest-calculator


3. Ensure you have Bash installed on your system (most Unix-based systems come with it pre-installed).

## Usage

To use the Simple Interest Calculator, run the following command in your terminal:


- `<principal>`: The principal amount (the initial sum of money).
- `<rate>`: The rate of interest (as a percentage).
- `<time>`: The time period (in years).
